# 買水還是找飲水機

背景：campus
主角：小澄
正確選項：1-2-A

### 引導對話
旁白｜你和小澄走到走廊。
旁白｜右邊是販賣機。
旁白｜左邊遠一點，是飲水機。[BREAK]
旁白｜販賣機比較近。
旁白｜飲水機要多走幾步。[BREAK]
小澄｜如果現在口渴，你會怎麼選？

### 選項
- A｜先找飲水機，用自己的水壺裝水
- B｜販賣機比較近，直接買瓶裝水
- C｜買兩瓶，等等可能會用到
- D｜看哪個包裝比較好看就買哪個

### 回應：A
旁白｜你往飲水機走去。
旁白｜小澄跟在旁邊。[BREAK]
旁白｜水流進水壺的聲音很輕。[BREAK]
小澄｜只是多走幾步
小澄｜就可以少製造一個瓶子[BREAK]
旁白｜你轉頭看她。
旁白｜她正好也看著你。
旁白｜兩個人的視線撞在一起。[BREAK]
旁白｜小澄很快移開眼神。[BREAK]
小澄｜...走、走吧，水快滿了。

### 回應：B、C、D
旁白｜小澄看著販賣機。
旁白｜她平常總是很溫柔。[BREAK]
旁白｜可是這一次，她少見地沉默了很久。[BREAK]
小澄｜...明明飲水機就在那邊
小澄｜明明只是多走幾步
旁白｜她抬頭看著你。[BREAK]
旁白｜聲音還是很輕。
旁白｜但語氣比剛才更顫。[BREAK]
小澄｜為什麼要買瓶裝水!!!!!!!![BREAK]
旁白｜這句話落下後，走廊突然安靜了。
旁白｜小澄像是也被自己的音量嚇到。
旁白｜她低下頭。[BREAK]
小澄｜...對不起
小澄｜我不是想兇你
旁白｜她握緊自己的水壺。[BREAK]
小澄｜我只是...不想讓這條回收線一開始就走向 Bad End
小澄｜只是很多浪費不是因為惡意
小澄｜只是因為太順手了
旁白｜她停了一下。[BREAK]
小澄｜可是太順手的選擇，常常也是最難改掉的[BREAK]

### 知識點
有補水設備時，優先使用水壺與飲水機，可以減少一次性瓶子的浪費。
