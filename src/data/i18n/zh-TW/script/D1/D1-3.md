# 真的需要買水時，該怎麼選？

背景：campus
主角：小澄
正確選項：1-3-C

### 引導對話
旁白｜操場邊很熱。
旁白｜附近沒有飲水機。
旁白｜販賣機前有人排隊。[BREAK]
小澄｜有時候，真的會需要買水
小澄｜永續不是叫人什麼都不能買[BREAK]
旁白｜她看著你。[BREAK]
小澄｜那這種時候，怎麼買比較好？[BREAK]

### 選項
- A｜直接買最大瓶，喝不完也沒關係
- B｜多買幾瓶備著，方便最重要
- C｜選擇自己真的需要的容量，喝完後正確回收
- D｜隨便買，喝不完就放著

### 回應：C
小澄｜嗯
小澄｜這樣就很好[BREAK]
小澄｜永續不是禁止消費
小澄｜而是先想清楚，自己是不是真的需要[BREAK]

### 回應：A、B、D
旁白｜小澄看著地上的空瓶。[BREAK]
小澄｜真正造成浪費的，有時候不是那一瓶水
小澄｜是明明不需要，卻還是習慣性地多買[BREAK]
旁白｜她看向你[BREAK]
小澄｜如果每一次都說『只是這一次』...
小澄｜那最後，就會變成每天都是這一次。[BREAK]

### 知識點
必要消費可以接受，重點是避免過度購買，並在使用後正確處理包裝。
