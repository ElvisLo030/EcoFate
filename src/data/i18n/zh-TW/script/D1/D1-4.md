# 喝完瓶裝水後怎麼處理？

背景：club
主角：小澄
正確選項：1-4-A

### 引導對話
旁白｜回到社團教室。
旁白｜小澄拿起一個空瓶模型。[BREAK]
小澄｜如果你真的買了一瓶水。
小澄｜也把它喝完了。
旁白｜她把瓶子放到你面前。[BREAK]
小澄｜那接下來呢？

### 選項
- A｜清空內容物，壓扁後投入正確回收分類
- B｜直接丟一般垃圾
- C｜裡面還有水也沒關係，直接丟回收
- D｜放在桌上，讓別人處理

### 回應：A
旁白｜你確認瓶子已經空了。
旁白｜然後把瓶身壓扁。[BREAK]
旁白｜喀啦。[BREAK]
小澄｜對[BREAK]
小澄｜如果包裝已經產生了[BREAK]
小澄｜就讓它有機會回到正確的循環裡[BREAK]

### 回應：B、C、D
旁白｜小澄伸手按住瓶子。[BREAK]
小澄｜等一下[BREAK]
小澄｜瓶子不是離開你的手，就算處理好了[BREAK]
旁白｜她看著你。[BREAK]
小澄｜你不是不懂
小澄｜只是想讓別人替你收尾，對嗎？[BREAK]

### 知識點
包裝已經產生後，清空、壓扁，讓它有機會回到正確的循環裡。
