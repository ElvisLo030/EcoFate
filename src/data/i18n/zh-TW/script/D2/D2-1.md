# 瓶內還有飲料時怎麼辦？

背景：campus
主角：可可
正確選項：2-1-C

### 引導對話
旁白｜可可晃了晃瓶子。
旁白｜裡面的飲料也跟著晃。[BREAK]
可可｜裡面還剩一點
可可｜這種狀態下，你要怎麼處理我？

### 選項
- A｜有飲料也沒關係，直接丟回收
- B｜直接丟一般垃圾
- C｜先把剩下的飲料倒掉，再準備回收
- D｜留在地上，反正會有人掃

### 回應：C
可可｜哼...還算知道一點嘛[BREAK]
旁白｜她嘴上這麼說，表情卻明顯放鬆了一點。[BREAK]
可可｜殘液會污染其他回收物
可可｜這一步，算你合格[BREAK]
旁白｜她別過臉。[BREAK]
可可｜...比我想像中細心一點[BREAK]

### 回應：A、B、D
可可｜你認真的？[BREAK]
旁白｜她皺起眉。[BREAK]
可可｜為什麼要把還有飲料的我丟出去啊！
可可｜你知道這樣會害其他回收物也被污染嗎？

### 知識點
寶特瓶回收前，應先清空內容物，避免殘液污染其他回收物。
