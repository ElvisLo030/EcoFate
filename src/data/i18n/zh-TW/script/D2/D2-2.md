# 是否壓扁、是否簡單清潔？

背景：campus
主角：coco
正確選項：2-2-A

### 引導對話
可可｜飲料處理完了[BREAK]
旁白｜她把瓶子遞給你。
旁白｜動作有點彆扭。[BREAK]
可可｜那接下來呢？
可可｜你不會就這樣把我丟進去吧？

### 選項
- A｜壓扁減少體積，再投入回收
- B｜不壓扁，直接丟進去就好
- C｜把瓶蓋、吸管、垃圾全部塞進瓶子裡
- D｜用完就丟一般垃圾比較快

可可｜喂！輕一點啦！[BREAK]
旁白｜她嘴上抗議。
旁白｜但沒有阻止你。[BREAK]
可可｜不過...這樣確實比較省空間
可可｜還不錯嘛，{playerName}[BREAK]

可可｜下次...也只能你來處理我喔
可可｜不...不是那個意思！我是說回收！[BREAK]

### 回應：B、C、D
可可｜你以為回收就是丟進去就好？
可可｜把麻煩丟給別人，不叫回收[BREAK]
旁白｜她瞪著你。[BREAK]
可可｜你真是高高在上呢，{playerName}
