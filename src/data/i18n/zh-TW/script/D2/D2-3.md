# 要不要查附近回收站

背景：campus
主角：可可
正確選項：2-3-A

### 引導對話
旁白｜可可雙手抱胸。[BREAK]
可可｜說得那麼好聽
可可｜那你知道附近哪裡可以回收嗎？
可可｜還是你只是嘴上說說？

### 選項
- A｜打開高雄永續生活消費引導平台，查附近回收站
- B｜不知道，所以算了
- C｜隨便找一個垃圾桶丟
- D｜拿回去放著，之後再說
