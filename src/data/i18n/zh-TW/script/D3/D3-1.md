# 鋁箔包能不能直接當紙類？

背景：club
主角：箔音
正確選項：3-1-B

### 引導對話
箔音｜回答我！[BREAK]
箔音｜你真的懂我的材質嗎？

### 選項
- A｜看起來像紙，就直接丟紙類
- B｜鋁箔包可能是複合包材，不能只靠外觀判斷
- C｜跟寶特瓶一起丟
- D｜直接丟一般垃圾最省事

### 回應：B
箔音｜...至少你沒有第一眼就把我歸類。[BREAK]
旁白｜她別過頭。[BREAK]
箔音｜我不是那麼簡單的材質。[BREAK]
旁白｜她停了一下。[BREAK]
箔音｜你願意先理解我，而不是急著把我分類。[BREAK]
箔音｜...這點，我不討厭。[BREAK]

### 回應：A、C、D
箔音｜看起來像紙，所以就是紙。[BREAK]
旁白｜她笑了一下。
旁白｜不是開心的笑。[BREAK]
箔音｜真方便。[BREAK]
箔音｜你真是高高在上呢，{playerName}。[BREAK]
箔音｜我的材質、我的去處，原來都由你一句「看起來像」決定。[BREAK]

### 知識點
複合包材不應只靠外觀分類，應先了解材質再決定去處。
