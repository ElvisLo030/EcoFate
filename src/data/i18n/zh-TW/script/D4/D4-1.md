# 妳是鐵還是鋁？

背景：recycling_area
主角：金穗
正確選項：4-1-C

引導對話
旁白｜金穗指著籃子裡兩種類似的罐子。
金穗｜{playerName}，雖然我們外表看起來差不多
金穗｜但如果不分清楚就進熔爐，會讓大家很困擾的[BREAK]
金穗｜你知道要怎麼最簡單地分辨「鐵罐」和「鋁罐」嗎？

選項
- A｜用力捏捏看，捏得動的就是鋁罐
- B｜看包裝顏色，亮的是鋁，暗的是鐵
- C｜使用磁鐵，會吸住的是鐵罐，不會吸的是鋁罐
- D｜喝喝看，味道不一樣的就是鐵罐

回應：C
金穗｜很聰明喔！磁鐵就是我們最好的測試機[BREAK]
旁白｜她露出讚許的微笑。[BREAK]
金穗｜鐵罐通常較堅硬，鋁罐則輕便且不吸磁
金穗｜分開處理，回收效率會更高喔[BREAK]

回應：A、B、D
金穗｜這可不是正確的判斷方式喔[BREAK]
旁白｜她輕輕搖了搖頭。[BREAK]

金穗｜下次試試看用磁鐵吧[BREAK]

知識點
鐵罐具磁性可被磁鐵吸引，鋁罐則無。正確區分材質可提升金屬回收的純度與循環價值。
