# 這是一場無限的轉世

背景：sunset_campus
主角：金穗
正確選項：4-3-A

引導對話
旁白｜金穗看著遠處的夕陽。
金穗｜小澄常羨慕我有一種「超能力」
金穗｜她們回收幾次後品質就會下降，但我卻可以一直持續下去[BREAK]
金穗｜{playerName}，你知道金屬回收最大的價值是什麼嗎？

選項
- A｜它可以無限次循環再生，且不會損失品質
- B｜因為它是金屬，所以賣掉比較貴
- C｜回收後的金屬可以直接變成人造衛星
- D｜金屬回收不需要消耗任何能源

回應：A
金穗｜這就是我的「無限轉世」[BREAK]
旁白｜她轉過頭，夕陽照在她金色的髮絲上。[BREAK]
金穗｜比起重新開採礦石，回收鋁罐可節省 95% 的能源
金穗｜只要正確分類，我就能永遠存在[BREAK]

回應：B、C、D
金穗｜雖然價值高是事實，但那不是核心喔[BREAK]
旁白｜她微微垂下眼簾。[BREAK]
金穗｜重點在於我們能不降級地反覆使用
金穗｜如果忽視了這一點，就太可惜了[BREAK]

知識點
金屬（尤其是鋁）可 100% 無限次循環回收，且不會像塑膠因回收次數增加而劣質化，是極具價值的永續資源。
