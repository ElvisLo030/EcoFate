# 數位的淨化程序

背景：server_room
主角：一雲
正確選項：5-3-A

引導對話
一雲｜{playerName}，別以為躲在雲端就沒有廢棄物。[BREAK]
一雲｜冗餘的廣告郵件、重複的截圖，伺服器的冷卻系統正在為這些數位垃圾耗能。[BREAK]
一雲｜你會如何優化你的「數位碳足跡」？

選項
- A｜定期清理不必要的郵件與數據，減少伺服器負擔
- B｜全部存在雲端硬碟，反正雲端空間是無限的
- C｜多申請幾個帳號分散存儲，這樣就不會滿了
- D｜把手機關機，這樣就不會產生數位碳足跡

回應：A
一雲｜優化完成。數據潔淨度提升，節能效率優化完成。[BREAK]
旁白｜她伸出手指，在空中做了一個「刪除」的動作。[BREAK]
一雲｜數位化不代表可以無止盡地浪費空間。[BREAK]
一雲｜隨手清理數據，也是一種看不見的永續行動。[BREAK]

回應：B、C、D
一雲｜偵測到低效率的數據管理思維。[BREAK]
旁白｜她發出輕微的數位雜訊聲，像是有些無奈。[BREAK]
一雲｜存儲數據需要電力，過多的數位負擔一樣會加重地球的壓力。[BREAK]
一雲｜請不要在我的雲端裡留下無意義的雜訊。[BREAK]

知識點
數據儲存於資料中心需消耗龐大電力。定期清理不必要的電子郵件、雲端備份，能有效減少「數位碳足跡」。
