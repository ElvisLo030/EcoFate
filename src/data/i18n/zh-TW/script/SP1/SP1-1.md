# 為什麼想買高級瓶裝水

背景：shop
主角：艾薇
正確選項：SP1-1-B

### 引導對話
艾薇｜{playerName}
艾薇｜你選擇我，是因為需要，還是因為品牌？

### 選項
- A｜因為看起來比較高級
- B｜先判斷自己是否真的需要，再考慮購買
- C｜價格越高越好，不需要想太多
- D｜別人買我也買

### 回應：B
艾薇｜你沒有立刻被品牌吸引
艾薇｜這很好
艾薇｜真正的品味，是知道自己為什麼選擇[BREAK]

### 回應：A、C、D
艾薇｜原來你看見的，只是標籤
艾薇｜你喜歡的，是我被擺在貨架上的樣子
艾薇｜至於我最後會去哪裡，對你來說並不重要[BREAK]

### 知識點
真正的品味，是知道自己為什麼選擇，而不是被品牌主導。
