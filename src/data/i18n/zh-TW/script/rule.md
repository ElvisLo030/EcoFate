# 遊戲流程

| 條件 | 結局 |
| ----------------------------------- | ---------------------------- |
| D2 ≥ 100 | 觸發 SP1（D2 結束後進入，結束後再進入 D3）|
| D1 + D2 + D3 + D4 ≥ 200   | 觸發 SP2（D4 結束後進入，結束後再進入 D5）|
| D1 + D2 + D3 + D4 + D5 ≥ 400 | Good Ending（SP1、SP2 分數不計入）|
| D1 + D2 + D3 + D4 + D5 ≥ 450 且 SP1 + SP2 ≥ 75 | True Ending（分數符合後 D5 直接進 TE）|
| 其他   | Bad Ending |

---
前導文字 -> D0 -> D1 -> D2 (如果D2 ≥ 100進入SP1否則進D3) -> D3 -> D4 -> (如果D1 + D2 + D3 + D4 ≥ 200進入SP2否則進D5) -> D5 -> ED

---

一般主線每天最高可得100分，共500分
隱藏劇情每次最高可得50分，最高100分
D5-4結束後直接依據分數進入ED
